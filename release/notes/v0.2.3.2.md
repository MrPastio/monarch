# Monarch v0.2.3.2

Первый release candidate со встроенным обновлением Monarch: проверка подписанного stable-канала при запуске, один клик для загрузки и установки, сохранение пользовательских данных, health acknowledgement и автоматический rollback при неудачном запуске.

Публикация остаётся закрыта до прохождения release acceptance и включения реального signing key.
