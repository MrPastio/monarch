# Monarch v0.2.3.2

Первый публичный Monarch со встроенным обновлением: приложение само проверяет подписанный stable-канал GitHub и официальный сайт, показывает доступную версию и устанавливает её в один клик с перезапуском.

- пользовательские данные, чаты, настройки и Monarch Safe не переустанавливаются и не удаляются;
- загрузка проверяется по Ed25519-подписи, размеру и SHA-256;
- новая версия ставится транзакционно, подтверждает исправный запуск и автоматически откатывается при сбое;
- центральный Oscar всегда остаётся на пустом чате, а после первого сообщения превращается в мини-маскота, которого можно скрывать кнопкой Monarch, перемещать и масштабировать.

Это одноразовый bootstrap-релиз: пользователям предыдущих версий нужно запустить этот установщик поверх текущей установки. Начиная с v0.2.3.2 следующие обновления приходят прямо внутри Monarch.
