# Monarch v0.2.3.4

Hotfix for the packaged Windows runtime.

- Fixed the installed release bundle failing before startup on Node built-ins used by bundled CommonJS dependencies.
- Added an executable release-bundle gate: installer assembly now stops unless the exact generated `monarch-server.mjs` can run.
- Preserved the complete `v0.2.3.3` product snapshot, including Agent Runtime V2 foundation, Research, long Voice/STT, composer and mascot behavior.

Users of `v0.2.3.2` or `v0.2.3.3` must run this installer once because those versions can fail before the in-app updater starts. Future releases continue through the in-app update flow.
