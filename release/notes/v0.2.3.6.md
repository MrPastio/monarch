# Monarch v0.2.3.6

Launcher state-machine hotfix for Monarch Desktop.

- Treats `committed` and `rollback-required` update transactions as terminal states.
- Prevents a healthy installed update from re-entering the candidate trial loop on later launches.
- Prevents a completed rollback from being repeated on every launch.
- Publishes launcher `1.0.1` and revokes `0.2.3.5` so affected installs are moved forward immediately.
- Includes every Oscar, Voice, updater-policy, and UI fix from `0.2.3.5`.

The update preserves existing configuration, chats, models, local data, and Monarch Safe storage.
